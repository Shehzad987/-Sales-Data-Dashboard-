# Sales Data Dashboard (Excel):

-- Overview:
 
An interactive Excel dashboard for analyzing sales data across time, categories, customers, and states using pivot tables, charts, and slicers.

--Features:

-Profit trends by category (Furniture, Office Supplies, Technology)  
-Customer count by year  
-Top 5 customers (profit contribution)  
-Monthly sales trends  
-Sales by state (pivot analysis)  
-Filters for Category and Year  

--Tools Used:

-Microsoft Excel  
-Pivot Tables & Charts  
-Slicers  

--Key Insights:

-Identifies top-performing categories and customers  
-Highlights regional (state-wise) sales performance  
-Reveals monthly and yearly trends
